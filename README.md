Project: Data Migration Validation Framework

Description:
Built a Python-based validation framework to compare datasets between source (MySQL) and target (Excel) systems.

Features:
- Detects missing and extra records
- Performs field-level comparison
- Identifies decimal precision differences
- Generates automated Excel reports
- Handles data type inconsistencies

Tech Stack:
- Python
- Pandas
- MySQL
- Excel

Use Case:
Simulates real-world data migration validation (SQL Server → PostgreSQL scenario)
